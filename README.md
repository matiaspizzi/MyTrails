# MyTrails - Personal project to learn Ruby and Rails

Video testing it out: [<https://www.youtube.com/watch?v=E_s-mRAsok8>](https://www.youtube.com/watch?v=vPSpGtzMLvg)

The idea was to have an application to set yearly or monthly goals that an employee would like to achieve, having a leader supervising the process and setting scores.

As an employee, you can:
- Register and log in to the app.
- List your current objectives.
- Set objectives with an estimated completion date.
- Modify an objectives' status (New/In Progress/In Review).
- Delete an objective.
- View objective details.
- View your account details.

As a leader, you can:
- Access the same features as employees.
- View and filter all your employees' objectives.
- Review your employees' "In Review" objectives, rate them, and mark them as "Done"
  
As an administrator, you can:
- Access the same features as leaders.
- View and filter all employee/leader objectives.
- View all role assignments.
- Assign a role to a user.
- Establish a leader/employee relationship between users.
